sheeldotme-duosexagesimal-system
================================

![build status](https://api.travis-ci.org/sheeldotme/sheeldotme-aba.svg)

Install
=======

  npm install sheeldotme-aba

Usage
=====
```javascript
  var aba = require('sheeldotme-aba');
  aba.toDecimal('1aA');
  aba.fromDecimal(100);
```
